# Tmux 快捷键大全

> 前缀键: `Ctrl+B` (配置中改为 `^A`，默认 tmux 是 `Ctrl+B`)
> 以下 `P` 代表前缀键，即先按 `Ctrl+A` 松开，再按后续键

---

## 一、Session 会话管理

| 快捷键 | 功能 |
|--------|------|
| `P + S` | 选择/切换 session (choose-session) |
| `P + ^N` | 新建 session |
| `P + o` | **SessionX** 模糊搜索会话 (插件 tmux-sessionx) |
| `P + P` | 发送前缀键到内层程序 (send-prefix) |
| `P + b` | 切换到上一个窗口 (last-window) |
| `P + d` / `P + ^D` | 脱离 (detach) 当前 session |
| `P + *` | 列出所有客户端 (list-clients) |

### SessionX 插件内快捷键 (在 SessionX 弹窗内)

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+y` | 在新窗口中打开选中的 zoxide 路径 |
| 上下箭头 / `Ctrl+j/k` | 在列表中导航 |
| `Enter` | 切换到选中的 session |
| `Ctrl+d` | 删除选中的 session |

---

## 二、Window 窗口管理

| 快捷键 | 功能 |
|--------|------|
| `P + ^C` | 新建窗口 (在 $HOME 目录) |
| `P + H` | 上一个窗口 (previous-window) |
| `P + L` | 下一个窗口 (next-window) |
| `P + ^A` | 最近使用的窗口 (last-window) |
| `P + r` | 重命名窗口 |
| `P + w` / `P + ^W` | 列出所有窗口 |
| `P + "` | 选择窗口 (choose-window) |
| `P + &` | 关闭窗口 (默认 tmux 键) |

---

## 三、Pane 面板管理

### 分屏

| 快捷键 | 功能 |
|--------|------|
| `P + s` | **竖向分屏** (上下分割，保留当前路径) |
| `P + v` | **横向分屏** (左右分割，保留当前路径) |
| `P + \|` | 竖向分屏 (不保留路径) |

### 面板导航 (Vim 风格)

| 快捷键 | 功能 |
|--------|------|
| `P + h` | 切换到 **左边** 面板 |
| `P + j` | 切换到 **下边** 面板 |
| `P + k` | 切换到 **上边** 面板 |
| `P + l` | 切换到 **右边** 面板 |

### 面板操作

| 快捷键 | 功能 |
|--------|------|
| `P + z` | 面板最大化/还原 (zoom toggle) |
| `P + c` | 关闭当前面板 (kill-pane) |
| `P + x` | 与下一个面板交换位置 (swap-pane) |
| `P + P` | 切换面板边框状态显示 |
| `P + *` | 同步输入到所有面板 (synchronize-panes) |

### 面板大小调整

| 快捷键 | 功能 |
|--------|------|
| `P + ,` | 向**左**扩展 20 列 (可重复按) |
| `P + .` | 向**右**扩展 20 列 (可重复按) |
| `P + -` | 向**下**扩展 7 行 (可重复按) |
| `P + =` | 向**上**扩展 7 行 (可重复按) |

---

## 四、复制模式 (Vi 风格)

> 按 `P + [` 进入复制模式

| 快捷键 | 功能 |
|--------|------|
| `v` | 开始选择 (begin-selection) |
| `y` | 复制选中内容到系统剪贴板 (tmux-yank 插件) |
| `q` | 退出复制模式 |
| `h/j/k/l` | 光标移动 |
| `/` | 向下搜索 |
| `?` | 帮助菜单 (list-keys)，按 `q` 退出 |
| `n/N` | 搜索下一个/上一个 |
| `Ctrl+u/d` | 半页上翻/下翻 |
| `g` | 跳到顶部 |
| `G` | 跳到底部 |

---

## 五、浮动窗口 (tmux-floax 插件)

| 快捷键 | 功能 |
|--------|------|
| `P + p` | 打开/关闭浮动终端 (80%x80% 大小) |

浮动窗口配置:
- 大小: 80% x 80%
- 边框颜色: magenta
- 文字颜色: blue
- 自动切换到当前路径

## 六、URL 与文本提取

### tmux-fzf-url 插件

| 快捷键 | 功能 |
|--------|------|
| `P + u` | 提取屏幕上所有 URL，用 fzf 选择并打开 |

### tmux-thumbs 插件

| 快捷键 | 功能 |
|--------|------|
| `P + Space` | 高亮屏幕上的可复制片段 (路径、hash、IP 等)，按提示键快速复制 |

### tmux-fzf 插件

| 快捷键 | 功能 |
|--------|------|
| `P + F` | 打开 tmux-fzf 菜单 (管理 session/window/pane/命令等) |

---

## 七、Session 持久化 (resurrect + continuum)

| 快捷键 | 功能 |
|--------|------|
| `P + Ctrl+s` | 手动保存当前所有 session 布局 |
| `P + Ctrl+r` | 手动恢复上次保存的 session |

> continuum 插件会自动定时保存，tmux 启动时自动恢复上次的 session。
> resurrect 配置了 nvim session 策略，会尝试恢复 nvim 的编辑状态。

---

## 八、其他实用快捷键

| 快捷键 | 功能 |
|--------|------|
| `P + R` | 重新加载 tmux 配置文件 |
| `P + K` | 清屏 (发送 `clear` + Enter) |
| `P + ^X` | 锁定服务器 |
| `P + ^L` / `P + l` | 刷新客户端显示 |
| `P + :` | 进入命令提示符 |

---

## 九、TPM 插件管理器

| 快捷键 | 功能 |
|--------|------|
| `P + I` | 安装配置中声明的新插件 |
| `P + U` | 更新所有已安装插件 |
| `P + Alt+u` | 卸载不在配置中的插件 |
